STORMON
-------

1. untar stormon.tar in the $EMDROOT directory

2. Insert the $EMDROOT/sysman/admin/scripts/stormon/stormon_metadata_host.xml into the file $EDMROOT/sysman/admin/metadata/host.xml

3. Insert the $EMDROOT/sysman/admin/scripts/stormon/stormon_collection_host.xml into the files $EDMROOT/sysman/admin/default_collection/host.xml

3.1 Insert the $EMDROOT/sysman/admin/scripts/stormon/stormon_collection_oracle_databse.xml into the files $EDMROOT/sysman/admin/default_collection/oracle_databse.xml

4. Include $EMDROOT/sysman/admin/scripts/stormon in path

4.1 Log in as root and execute the root.sh script in $EMDROOT/sysman/admin/scripts/stormon directory

5. Stop and Start or restart the em agent
